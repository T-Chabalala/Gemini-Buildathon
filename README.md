# Citizen Service Agent Demo

A lightweight prototype that routes citizen queries to the right department for home affairs, SASSA, or municipal complaints, and shows a simple status update.

## Run locally

Open [public/index.html](public/index.html) in a browser, or serve the folder with a simple static server.

Example:

```bash
python -m http.server 8000 --directory public
```

Then open http://localhost:8000/.
